Objetivo

Este manual apresenta todas as funcionalidades disponíveis para o Administrador do El Prado OS.

⸻

LOGIN

O administrador acessará:

admin.html

Será necessário informar:

• Usuário

• Senha

Após autenticação, será criada uma sessão com duração máxima de 1 hora de inatividade.

⸻

• Clientes cadastrados

SEGURANÇA

Após 1 hora de inatividade, o sistema encerrará automaticamente a sessão.

Todas as ações administrativas serão registradas.

⸻

BOAS PRÁTICAS

Não compartilhar usuário e senha.

Realizar logout ao finalizar o uso.

Cadastrar produtos com imagens em alta qualidade.

Manter promoções atualizadas.

Conferir pedidos frequentemente.

Acompanhar o Dashboard diariamente.

⸻

OBJETIVO

Permitir que o administrador controle toda a operação da hamburgueria através de uma única plataforma.
