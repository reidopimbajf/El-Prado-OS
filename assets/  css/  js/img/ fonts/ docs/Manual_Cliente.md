👤 MANUAL DO CLIENTE

Projeto: EL PRADO OS

Versão: 0.7 Enterprise

Última atualização: 17/06/2026

⸻

Objetivo

Este manual apresenta todas as funcionalidades disponíveis para os clientes da El Prado Burguer.

O objetivo é proporcionar uma experiência simples, rápida e intuitiva durante todo o processo de compra.

⸻

PRIMEIRO ACESSO

Ao acessar o site o cliente poderá:

• Navegar pela Home

• Conhecer a hamburgueria

• Visualizar promoções

• Ver produtos em destaque

• Acessar o cardápio

Não será necessário realizar login para navegar.

O login será solicitado apenas para finalizar um pedido.

⸻

CADASTRO

O cliente poderá criar sua conta informando:

Nome

Telefone

E-mail

Senha

Confirmação da senha

Após o cadastro será realizado login automaticamente.

⸻

LOGIN

O acesso poderá ser realizado utilizando:

• E-mail

ou

• Telefone

Será obrigatório informar a senha cadastrada.

⸻

Banner principal

Finalizar pedido

SEGURANÇA

Os dados pessoais serão utilizados apenas para operação da hamburgueria.

As informações não poderão ser compartilhadas com terceiros.

⸻

SUPORTE

Caso o cliente encontre qualquer problema poderá entrar em contato através do WhatsApp oficial da hamburgueria.

OBJETIVO

Proporcionar uma experiência moderna, intuitiva e agradável desde o primeiro acesso até a entrega do pedido.
