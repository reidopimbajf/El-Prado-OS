👨‍🍳 MANUAL DO FUNCIONÁRIO

Projeto: EL PRADO OS

Versão: 0.7 Enterprise

Última atualização: 17/06/2026

⸻

Objetivo

Este manual orienta todos os funcionários responsáveis pela operação diária da hamburgueria.

⸻

PERFIS

O sistema possuirá os seguintes perfis:

Administrador

Gerente

Atendente

Cozinha

Entregador (Versão futura)

Cada perfil visualizará apenas as funcionalidades autorizadas.

⸻

LOGIN

Cada funcionário possuirá:

Usuário

Senha

Perfil de acesso

⸻

PEDIDOS

Ao iniciar o expediente:

Consultar novos pedidos.

Aceitar pedidos.

Encaminhar para cozinha.

Atualizar status.

Concluir pedidos.

⸻

COZINHA

A tela da cozinha apresentará:

Número do pedido

Produtos

Observações

Tempo de preparo

Status

Ao finalizar:

Alterar para:

Pedido Pronto.

⸻

ENTREGA

Após preparo:

Alterar para:

Saiu para entrega.

Quando entregue:

Concluir pedido.

⸻

CANCELAMENTO

Somente Administrador ou Gerente poderão cancelar pedidos.

Todo cancelamento deverá possuir justificativa.

⸻

CLIENTES

O funcionário poderá visualizar:

Nome

Telefone

Histórico

Observações importantes

Não poderá alterar informações pessoais do cliente.

⸻

PRODUTOS

Funcionários poderão apenas consultar.

Alterações serão realizadas somente por Administradores.

⸻

BOAS PRÁTICAS

Conferir observações dos pedidos.

Atualizar status imediatamente.

Evitar atrasos.

Manter comunicação com o cliente.

Registrar problemas encontrados.

⸻

SEGURANÇA

Nunca compartilhar senha.

Encerrar sessão ao final do expediente.

Comunicar acessos indevidos.

⸻

OBJETIVO

Garantir uma operação organizada, rápida e padronizada em toda a hamburgueria.
