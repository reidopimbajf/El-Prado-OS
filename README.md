# 📖 Sobre o Projeto

O El Prado OS é um sistema completo de gestão para hamburguerias, desenvolvido para oferecer uma experiência premium ao cliente e uma gestão eficiente para administradores.

O projeto contempla:

- 🌐 Site Institucional
- 🍔 Cardápio Digital
- 🛒 Carrinho Inteligente
- 💳 Checkout
- 👤 Área do Cliente
- 🎁 Programa de Fidelidade
- 📦 Gestão de Pedidos
- 📊 Dashboard Administrativo
- 📈 Relatórios Gerenciais
